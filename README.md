## üöÄ Getting Started

### Prerequisites
- Node.js 18.0 or higher
- pnpm, npm, or yarn

### Installation

1. **Clone and install:**
    ```bash
    git clone https://github.com/your-username/zebra-board.git
    cd zebra-board
    npm install
    ```

2. **Generate themes:**
    ```bash
    npm run generate-themes
    ```

3. **Start development server:**
    ```bash
    npm run dev
    ```

4. **Open [http://localhost:3000](http://localhost:3000)**
